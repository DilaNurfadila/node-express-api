# Node Express API

#### How to install

- `npm init -y`

- `npm install --save express`

- `npm install --save-dev nodemon`

- `npm install uuid`

#### How to run this app

- `node index.js` (start without live run every saved)

or

- `npm start` (start with live run every saved)
